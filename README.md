# 🛒 Market Basket Analysis

## 📌 Overview

This project uses the Apriori Algorithm to identify products that are frequently purchased together. The analysis helps businesses improve cross-selling, product placement, and promotional strategies.

## 🎯 Objectives

* Discover frequent itemsets
* Generate association rules
* Analyze product relationships
* Provide business recommendations

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Mlxtend
* Matplotlib

## 📂 Dataset

Online Retail Dataset

## 📈 Results

* Identified frequently purchased product combinations.
* Generated association rules using Support, Confidence, and Lift.
* Provided insights for product bundling and recommendation systems.
